HTML Select Example

This page contains a few examples of how to use an HTML dropdown (select) box to link to other pages. Since this is a misuse of the select element, we have to cheat by using a tiny bit of JavaScript to do the dirty work.

You may examine source of this page to see how it works, or you should be able to right-click and "View Source" (since Internet Explorer rudely ignores the HTTP Content-Type).
